# Accelerator PowerPoint source review

The two uploaded decks have been mapped for conversion into dedicated interactive accelerator content.

## Teaching cycle preserved

The decks use the same strong cycle throughout:

1. try the task first;
2. allow retrieval/thinking time;
3. use a video only if still stuck;
4. explain the method or retrieve related flashcards;
5. retry;
6. self-rate and revisit through spacing.

This is now reflected in the Exam Accelerator dashboard panel in v8.2.1.

## Python deck

- 28 practical Python task slides were identified.
- 27 task-to-video links were identified.
- 20 tasks include a mapped Haggis pseudocode comparison.
- The skills progress from output/input and data types through selection, loops, running totals, validation, arrays, random, `len`, `round` and Boolean variables.
- Some Python answer slides store the answer as an image rather than editable text. They should be rebuilt as live code activities rather than embedded as screenshots.

## SQL deck

- 33 linked practice-and-answer tasks were identified.
- Coverage includes SELECT/FROM, WHERE, AND/OR, single and multiple ORDER BY fields, equi-joins, INSERT, UPDATE and DELETE.
- The deck says its videos are not ready, so the app should use concise in-app teaching and worked examples until SQL videos are supplied.

## Recommended conversion

Each source task should become an app activity with:

- an editable Python or SQL response area;
- a three-minute optional retrieval timer;
- a **Show explanation/video** button;
- a **Compare with Haggis** panel for mapped Python tasks;
- automatic or structured checking;
- the four confidence choices already used in the decks;
- a retry before the model answer is revealed;
- spaced review scheduled from the confidence choice.

The machine-readable extraction is in `accelerator-source-map.json`.
