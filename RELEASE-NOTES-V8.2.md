# Nat 5 Computing v8.2.0

## Added

- Nat 5 Computing branding on the live entry page.
- Separate individual learner email/password sign-in.
- Individual password-reset email button.
- Permanent complimentary entitlement display for founding teachers and pupils.
- Entitlement validation before an individual pupil can enter the course.
- Full Course and Exam Accelerator route cards.
- Saved learning-route preference.
- Rapid accelerator pathway generated from the existing shared activity bank.
- Individual learning scope for progress, games, revision and spaced practice.
- Course and route metadata on new progress records.
- Independent pupils are excluded from class-only custom content and peer marking.
- Updated Firestore rules for active individual entitlements.

## Preserved

- Existing teacher invitations and Google sign-in.
- Existing class codes and school pupil accounts.
- Existing class progress document IDs.
- Teacher dashboards, custom activities and anonymous in-class peer review.

## Not included yet

- Public individual registration or checkout.
- Stripe payment handling.
- Cloud Functions.
- Paid licence renewal/refund automation.
- A standalone baseline diagnostic exam.

These remain suitable for the Blaze/paid-launch phase.
